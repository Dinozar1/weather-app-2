Detailed Description

Provides class for storing Station information and geolocation features.

Provides struct for Station data and struct for geolocation data, provides methods for fetching api and parsing data to json format and methods for geolocation purposes

Callback function for libcurl to handle received data.

When libcurl makes an HTTP request, data isn't received all at once. It comes in chunks. This callback function:

  • Collects these chunks
  • Stores them in a single string
  • Allows processing the entire response
    Parameters
    contentsPointer to the received data
    sizeSize of each data element
    nmembNumber of data elements
