This application allows users to\+:
\begin{DoxyItemize}
\item Fetch all measurement stations across Poland
\item Filter stations by city name
\item Search for stations within a given radius using Open\+Street\+Map\textquotesingle{}s Nominatim geocoding service
\item View measurement sensors for selected stations
\item Display and analyze measurement data with statistical processing and graphical representation
